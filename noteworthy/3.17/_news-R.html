Upgrade to Eclipse Jetty 10.0 in RAP Equinox target and in RAP Tools

The Eclipse Platform and the 2021-06 Simultaneous Release updated from Eclipse Jetty 9.4 to version 10.0. Eclipse RAP follows this upgrade in order to align its dependencies with those of other projects, and distributes the relevant Jetty bundles in its RAP Equinox target for RAP Runtime, and in RAP Tools.

Increase the minimum required Servlet API to 4.0

With the upgrade of the included Eclipse Jetty parts, the minimum required Servlet API level is incremented from 3.1 to 4.0. The servlet API bundle is now contributed by the Jetty project, and distributed as part of the RAP Equinox feature in RAP Runtime.
