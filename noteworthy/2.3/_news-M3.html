Breaking changes in Firefox 29+

Some browser specific API, used for detection of Firefox, have been removed from Firefox 29. As a result, RAP application may not start (RAP <= 2.1) or may work with visual glitches (RAP 2.2) . This issue is fixed in M3. For older versions of RAP, please visit the FAQ for workarounds that you can apply yourself.

Migrated DropDown widget to RAP Core

The DropDown widget has been graduated from the incubator and is now included in the RAP core.

Client-side (C)Combo rework

The JavaScript part of the Combo widget implementation has been completely rewritten. It now uses client-side DropDown implementation for it's list, allowing as to handle huge amonth of items without performance degradation. Now it's also possible the theme Combo list items independently. Combo-List-Item CSS element supports color, background-color, background-image, text-decoration, text-shadow and padding properties.
