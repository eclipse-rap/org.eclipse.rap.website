Theming

Theme Extensions

The theming was extended in serveral places:

  • ProgressBar has a new themeable property width.
  • box-shadow property has been added to buttons.

Application Context

Application Context Listener

ApplicationContextListeners can be attached to an application context to receive a notification before the application context is destroyed.

ClientScripting

The ClientScripting has been graduated and now it's included in the RAP core. ClientScripting allows developers to handle some of the events directly on the client, without creating any http-requests. This is ideal to customize or enhance the behavior of specific widgets, making it much less often necessary to develop RAP custom-widgets.

More information about ClientScripting can be found here.

The ClientScripting incubator projects are no longer compatible with RAP 2.2M1+ and must be used only with older RAP versions.

Widget Set

Synchronize user data with the client

It's now possible to transfer the data attached to the SWT widget to the client. To do so, the key for that data has to be registered with WidgetUtil.registerDataKeys( String... ). Example: 

WidgetUtil.registerDataKeys( "foo" );
widget.setData( "foo", "bar" );
